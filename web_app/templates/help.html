  • https://example.com/10k/microsoft-2024.html

Replace these with real 10‑K HTML links. PDFs are not supported.

What kind of URLs work best? Direct links to the 10‑K HTML filing. Avoid PDFs; the extractor needs HTML tables.
Can I use this with PDFs? No—please provide the HTML version of the filing so tables can be parsed.
How is OpenAI used? We always use OpenAI to map line items to the chart of accounts and to assess forward-looking revenue outlook; if unavailable, we fall back to deterministic mapping.
How does AI sentiment work? The model reads the 10-K and, optionally, recent news, assesses the forward-looking revenue outlook, and can adjust growth assumptions by up to ±1.5 pts.
Where are my files processed? On this service. The server binds to 127.0.0.1:8501 by default (private); nothing is sent elsewhere beyond the configured LLM provider.
